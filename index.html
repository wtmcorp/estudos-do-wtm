SEMANA 1–2

Excel Avançado Focado em Dados

Domine as ferramentas essenciais que todo analista de dados precisa

01

Tabela Dinâmica

📖 O que estudar:

  • Como transformar dados brutos em tabelas dinâmicas
  • Filtros, segmentação de dados e campos calculados
  • Agrupamento por datas (mês, trimestre, ano)
  • Atualização automática de tabelas dinâmicas

🎯 Exercício prático:

Pegue um dataset de vendas (pode baixar no Kaggle) e crie uma tabela dinâmica que mostre vendas por mês, por produto e por região. Adicione segmentação de dados.

02

PROCV / XLOOKUP

📖 O que estudar:

  • PROCV: sintaxe, erros comuns, limitações
  • XLOOKUP: busca bidirecional, valores padrão
  • Quando usar cada uma
  • Combinação com CORRESP e ÍNDICE

🎯 Exercício prático:

Crie duas planilhas: uma de pedidos e outra de clientes. Use PROCV para trazer o nome do cliente pelo ID. Depois refaça com XLOOKUP.

03

Análise de Dataset

📖 O que estudar:

  • Como explorar dados: estatísticas descritivas
  • Identificar padrões e outliers
  • Gráficos: barras, linhas, dispersão, combinados
  • Formatação condicional para destacar insights

🎯 Exercício prático:

Baixe um dataset do Kaggle (ex: Superstore Sales). Analise tendências de vendas, identifique os melhores produtos e crie 3 gráficos profissionais.

04

Limpeza de Dados

📖 O que estudar:

  • Remover duplicatas e espaços extras (ARRUMAR/TRIM)
  • Padronizar texto (MAIÚSCULA, MINÚSCULA, PRI.MAIÚSCULA)
  • Tratar valores em branco e #N/D
  • Texto para Colunas, Flash Fill
  • Validação de dados

🎯 Exercício prático:

Pegue um dataset "sujo" (com espaços, duplicatas, formatos errados) e limpe completamente usando as funções aprendidas.

SEMANA 3–4

Power BI

Transforme dados em dashboards interativos e impressionantes

01

Fundamentos e Interface

📖 O que estudar:

  • Interface do Power BI Desktop (3 visões)
  • Power Query: importar e transformar dados
  • Tipos de dados e formatação
  • Publicação no Power BI Service
02

Modelagem de Dados

📖 O que estudar:

  • Tabelas fato vs tabelas dimensão
  • Relacionamentos 1:N e N:N
  • Tabela calendário (dCalendario)
  • Boas práticas de modelagem
03

Dashboards

📖 O que estudar:

  • Visuais: cartões, gráficos, mapas, tabelas
  • Filtros e segmentação de dados (Slicers)
  • Drill-down e drill-through
  • Design: cores, alinhamento, hierarquia visual
  • Bookmarks e navegação entre páginas
04

DAX e Métricas

📖 O que estudar:

  • Medidas vs Colunas Calculadas
  • SUM, AVERAGE, COUNT, DISTINCTCOUNT
  • CALCULATE, FILTER, ALL
  • Inteligência temporal: TOTALYTD, SAMEPERIODLASTYEAR
  • FORMAT e funções de texto
SEMANA 5–6

SQL Básico

Consulte bancos de dados com confiança — o essencial que você precisa

01

SELECT e Filtros

📖 O que estudar:

  • SELECT *, SELECT colunas específicas
  • WHERE com operadores: =, !=, >, <, BETWEEN, IN, LIKE
  • ORDER BY ASC/DESC
  • LIMIT e OFFSET
  • Alias com AS

🎯 Onde praticar:

Use o SQLite Online ou DB Fiddle para praticar sem instalar nada.

02

JOIN

📖 O que estudar:

  • INNER JOIN: combinar tabelas pela chave
  • LEFT JOIN: manter todos da tabela esquerda
  • RIGHT JOIN e FULL OUTER JOIN
  • JOIN com múltiplas condições

🎯 Exercício prático:

Crie tabelas de pedidos e clientes. Faça JOIN para listar pedidos com nome do cliente. Teste LEFT JOIN para ver clientes sem pedidos.

03

GROUP BY e Agregações

📖 O que estudar:

  • GROUP BY com COUNT, SUM, AVG, MIN, MAX
  • HAVING vs WHERE
  • Agrupar por múltiplas colunas
  • Combinar GROUP BY com JOIN
04

Consultas Avançadas

📖 O que estudar:

  • Subqueries no WHERE e no FROM
  • CASE WHEN para criar categorias
  • COALESCE e NULLIF
  • Funções de data: YEAR, MONTH, DATEDIFF
SEMANA 7–8

3 Projetos Fortes

Projetos reais ligados à sua experiência — o que vai te destacar

PROJETO 1

Dashboard de Performance Logística

📊 Métricas obrigatórias:

🛠️ Como construir:

  • Simule dados de 6 meses de operação logística
  • Crie no Excel e depois replique no Power BI
  • Inclua filtros por período, transportadora e status
  • Adicione KPIs: SLA atingido, taxa de atraso, tempo médio
PROJETO 2

Análise de Estoque

📊 Métricas obrigatórias:

🛠️ Como construir:

  • Crie dataset com SKUs, entradas, saídas e saldo
  • Calcule giro = vendas / estoque médio
  • Sinalize produtos com estoque abaixo do mínimo
  • Monte curva ABC (80/15/5) com Pareto
PROJETO 3

Dashboard Financeiro de Operação

📊 Métricas obrigatórias:

🛠️ Como construir:

  • Combine dados de vendas, custos e frete
  • Calcule margem = (receita - custo) / receita
  • Crie mapa de calor por região
  • Dashboard com comparativo mês a mês
